# Usage
<!-- usage -->
```sh-session
$ npm install -g put-cli-tool
$ put COMMAND
running command...
$ put (-v|--version|version)
put-cli-tool/0.0.0 darwin-x64 node-v13.11.0
$ put --help [COMMAND]
USAGE
  $ put COMMAND
...
```
